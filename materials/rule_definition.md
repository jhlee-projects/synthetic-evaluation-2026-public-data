# Rule Definition

본 문서는 가계동향조사 원자료에 내재된 변수 간 제약조건을 정리한 것이다.

---

## 1. 가구관련

- 가구원수 ≥ 취업인원수

- 가구구분코드: '근로자가구'
  - 취업인원수 ≥ 1

- 배우자코드: '있음'
  - 가구원수 ≥ 2

- 노인가구여부: '해당'
  - 가구주 연령 ≥ 64
  - 세대구분코드: '1세대' 또는 '2세대'

- 모자가구여부: '해당'
  - 가구주 성별코드: '여자'
  - 세대구분코드: '2세대'
  - 배우자코드: '있음' 또는 '없음'
  - 가구원수 ≥ 2

- 맞벌이가구여부: '해당'
  - 배우자코드: '있음'

- 일반가구여부: '해당'
  - 노인가구여부: '아님'
  - 모자가구여부: '아님'
  - 맞벌이가구여부: '아님'

---

## 2. 가구주 관련

- 가구주 학력코드: '무학'
  - 가구주 수학구분코드: '무학'

- 가구구분코드: '근로자가구'
  - 가구주 취업여부: '취업'

- 가구주 취업여부: '비취업'
  - 가구주 7차직업분류코드: '기타/무직'
  - 가구주 종사상지위코드: NA

- 가구주 7차직업분류코드: '직업군인'
  - 가구주 종사상지위코드: '상용근로자'

---

## 3. 주거 및 자산

- 입주형태코드: '자기집'
  - 전세보증금 = 0
  - 월세사글세 = 0
  - 주택소유유무: '있음'

- 입주형태코드: '무상주택'
  - 전세보증금 = 0
  - 월세사글세 = 0

- 입주형태코드: '전세'
  - 월세사글세 = 0

- 입주형태코드: '보증금 없는 월세'
  - 전세보증금 = 0

---

## 4. 가계소득

- 가구구분코드: '근로자가구'
  - 전국근로자외가구 1인이상 소득분위코드: NA
  - 도시근로자외가구 소득분위: NA

- 가구구분코드: '근로자외가구'
  - 전국근로자가구 1인이상 소득분위코드: NA
  - 도시근로자가구 소득분위: NA

- 전국전체가구 소득분위 - 1 ≤ 전국근로자가구 소득분위 ≤ 전국전체가구 소득분위

- 전국전체가구 소득분위 ≤ 전국근로자외가구 소득분위 ≤ 전국전체가구 소득분위 + 1

- 전국전체가구 소득분위 ≤ 도시전체가구 소득분위 ≤ 전국전체가구 소득분위 + 1

- 도시읍면부구분: '비도시'
  - 도시전체가구 소득분위: NA

- 도시전체가구 소득분위 - 1 ≤ 도시근로자가구 소득분위 ≤ 도시전체가구 소득분위

- 도시전체가구 소득분위 ≤ 도시근로자외가구 소득분위 ≤ 도시전체가구 소득분위 + 1

---

## 5. 도시 구분

- 도시읍면부구분: '비도시'
  - 도시전체가구 소득분위: NA

---

## 6. 가계지출

- 가계지출금액 = 가계지출 소비지출금액 + 가계지출 비소비지출금액

- 가계지출 소비지출금액 = 소비지출금액 분류 총합
